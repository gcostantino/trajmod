# GNSS Trajectory Modeling Library

A refactored, production-ready library for modeling GNSS displacement trajectories with slow slip events (SSE) and earthquake signals.

## Features

- ✅ **Modular Design**: Clean separation of concerns
- ✅ **Type Hints**: Full type annotation throughout
- ✅ **No Hardcoded Paths**: Fully portable
- ✅ **Multiple Fitting Strategies**: OLS, LASSO, Elastic Net, Iterative
- ✅ **Event Catalogs**: SSE and earthquake management
- ✅ **Extensible**: Easy to add new templates and fitters
